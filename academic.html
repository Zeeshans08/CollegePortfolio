Academic Programs

Comprehensive, industry-relevant education to build successful careers in technology and beyond

Bachelor of Science in Information Technology

Program Overview

Our 3-year BSc IT program provides students with comprehensive knowledge of information technology concepts, software development, and system administration. The curriculum is designed to meet industry demands while providing strong theoretical foundations.

Key Features:

  • Industry-aligned curriculum updated regularly
  • Hands-on training with modern technologies
  • Industry certifications integrated into coursework
  • Dedicated placement cell for career support

Curriculum Structure

Semester I

  • Programming Fundamentals using Python
  • Discrete Mathematics
  • Computer Organization and Architecture
  • Digital Electronics
  • Communication Skills

Semester II

  • Data Structures and Algorithms
  • Database Management Systems
  • Operating Systems
  • Computer Networks
  • Environmental Studies

Semester III

  • Object Oriented Programming with Java
  • Web Programming
  • Software Engineering
  • Computer Graphics
  • Numerical Methods

Bachelor of Science in Computer Science

Program Overview

The BSc Computer Science program offers a rigorous foundation in computer science theory, algorithms, and system design. Students gain expertise in software development, data structures, and computational thinking.

Key Features:

  • Strong emphasis on problem-solving and algorithmic thinking
  • Advanced coursework in AI and Machine Learning (optional)
  • Research opportunities with faculty members
  • Regular hackathons and coding competitions

Curriculum Structure

Semester I

  • Introduction to Programming (C Language)
  • Calculus and Linear Algebra
  • Digital Electronics and Logic Design
  • Principles of Management
  • Communication Skills

Semester II

  • Advanced Data Structures
  • Computer Organization and Architecture
  • Discrete Mathematics
  • Probability and Statistics
  • Environmental Studies

Semester III

  • Design and Analysis of Algorithms
  • Database Management Systems
  • Operating Systems
  • Theory of Computation
  • Computer Networks
